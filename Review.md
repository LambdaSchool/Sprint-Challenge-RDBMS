# Questions - Self Study

You can exercise your Google-Fu for this and any other _Sprint Challenge_ in the future.

1.  Explain the difference between `RDBMS` and `SQL`.
    -- RDBMS is the the actual collection of data stored in a table while SQL is the language used to manage the data.

1.  Why do tables need a `primary key`?
    -- This is needed so that the information can be indentified specifically.

1.  What is the name given to a table column that references the primary key on another table.
    -- foreign key

1.  What do we need in order to have a _many to many_ relationship between two tables.
    -- Two tables that have something connect in many ways. Like a book written by many authors and the authors write many books.

1.  What SQL statement is used to retrieve data from a table?
    -- select

1.  What SQL clause is used to filter the results of a query?
    -- where
