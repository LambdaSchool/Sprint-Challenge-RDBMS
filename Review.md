# Questions - Self Study

You can exercise your Google-Fu for this and any other _Sprint Challenge_ in the future.

1.  Explain the difference between `RDBMS` and `SQL`.
    - RDBMS means Relational Database Management System it's like the environment that your database lives in and SQL is the Structured         Query(ing) Language you use to communicate with your database.

2.  Why do tables need a `primary key`?
    - They function as identifiers allowing you to quickly pull rows of information, and they also promote compartmentalizing information.
    
3.  What is the name given to a table column that references the primary key
    on another table.
    - A foreign key
    
4.  What do we need in order to have a _many to many_ relationship between two
    tables.
    - Another table to manage all the relationships.
    
5.  What SQL statement is used to retrieve data from a table?
    - SELECT
    
6.  What SQL clause is used to filter the results of a query?
    - WHERE
